Kevin Ninsiima — Portfolio

This is a simple static portfolio site built with HTML, CSS and JavaScript.

Files:

- index.html — main page
- css.css — styles
- java.js — small JS for typing effect, smooth scroll, and contact form stub

How to view locally

1. Open index.html in your browser (double-click or right-click → Open with).
2. Or serve it with a simple local server (recommended for consistent routing):

```powershell
python -m http.server 8000
```

Deploy to GitHub Pages

1. Create a GitHub repository and push these files.
2. In the repository settings → Pages, select the main branch and root folder.
3. Save and wait a minute — your site will be available at https://<your-username>.github.io/<repo-name>/

Notes

- Replace your-photo.jpg with your real profile photo (same folder) and update contact details in index.html.
- The contact form is a front-end stub (no backend). For production, connect it to an email service or form backend.
